# AVR_Frogger
A game made for the ATMega324A MCU. Includes LCD, Joystick, and Buzzer. Dodge the obstacles!

Using a slightly modified Nokia3110 (PCD8544) LCD library: https://github.com/LittleBuster/avr-nokia5110
